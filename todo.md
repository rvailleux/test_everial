## Use Cases

- [ ] **UC1 — Vérification d'identité (CNI / Passeport)**
  Documents : carte d'identité, passeport
  Extraction : nom, prénom, date de naissance, date d'expiration, MRZ

- [ ] **UC2 — Justificatif de domicile**
  Documents : facture EDF/eau/télécom, avis d'imposition
  Extraction : adresse complète

- [ ] **UC3 — Justificatif de revenus**
  Documents : bulletin de salaire, avis d'imposition
  Extraction : revenus, employeur

- [ ] **UC4 — RIB / Coordonnées bancaires**
  Documents : RIB
  Extraction : IBAN, BIC

- [ ] **UC5 — Permis de conduire**
  Documents : permis de conduire
  Extraction : données conducteur, catégories

- [ ] **UC6 — Carte vitale**
  Documents : carte vitale
  Extraction : numéro de sécurité sociale

- [ ] **UC7 — Extrait KBIS**
  Documents : extrait KBIS
  Extraction : SIRET, SIREN, raison sociale

- [ ] **UC8 — Titre de séjour**
  Documents : titre de séjour
  Extraction : identité, date de validité

- [ ] **UC9 — Vérification biométrique**
  Documents : CNI + selfie vidéo
  Extraction : score de similitude, détection du vivant

- [ ] **UC10 — Neurolens (extraction zero-shot)**
  Documents : n'importe quel document
  Extraction : clés/valeurs auto-détectées sans paramétrage

- [ ] **UC11 — Classification de courrier entrant**
  Documents : documents non structurés (courriers, factures, contrats…)
  Extraction : type de document détecté

- [ ] **UC12 — Détection de signature**
  Documents : n'importe quel document signé
  Extraction : signé / non signé, localisation

- [ ] **UC13 — Anonymisation de document**
  Documents : n'importe quel PDF
  Extraction : PDF caviardé en sortie
